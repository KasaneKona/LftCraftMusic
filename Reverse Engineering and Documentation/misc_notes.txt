Frequency Registers
----
for f = frequency in HZ,
f = 111860.7954375/(c_freq+1)
c_freq = (111860.7954375/f)-1

Track Table Decoding
----
TrackNo/2 = index of 3-byte struct
TrackNo LSB = second half?
If first half:
  Track pointer = Nybbles 102xxx
If second half:
  Track pointer = Nybbles xxx210
